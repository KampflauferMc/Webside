Werde Teil Unseres Server Teams

Builder

Ihre Aufgaben: Sie bauen als Builder bei uns die Maps für unsere User. Zudem sollten sie regelmäßig auf unser Server Netzwerk kommen. Das erwarten wir von ihnen: Sie haben ein Mikrofon und Lautsprecher und sind bereit die Software Teamspeak zu nutzen und scheuen sich nicht davor mit uns auf unserem Teamspeak Server zu sprechen. Außerdem sollten sie der deutschen Sprache im wörtlichen und schriftlichen mächtig sein. Sie sollten langjährige Erfahrung in Minecraft mitbringen. Zudem sollten sie freundlich und hilfsbereit sein.

Developer

Ihre Aufgaben: Sie Programmieren in Java Minecraft Plugins. und sollten regelmäßig auf unser Server Netzwerk kommen Das erwarten wir von ihnen: Sie haben ein Mikrofon und Lautsprecher und sind bereit die Software Teamspeak zu nutzen und scheuen sich nicht davor mit uns auf unserem Teamspeak Server zu sprechen. Außerdem sollten sind der deutschen Sprache im wörtlichen und schriftlichen mächtig sein. Sie sollten langjährige Erfahrung in Minecraft mitbringen. Zudem sollten sich in der Programmiersprache Java auskennen. Wichtig ist auch das sie sich mit GitHub auskennen. Als letztes sollten sie freundlich und hilfsbereit sein.

Supporter

Ihre Aufgaben: Sie helfen unseren Nutzern, indem sie Fragen rund um Minecraft und unseren Servern beantworten. Zudem sollten sie regelmäßig auf unser Server Netzwerk kommen. Das erwarten wir von ihnen: Sie haben ein Mikrofon und Lautsprecher und sind bereit die Software Teamspeak zu nutzen und scheuen sich nicht davor mit uns auf unserem Teamspeak Server zu sprechen. Außerdem sollten der deutschen Sprache im wörtlichen und schriftlichen mächtig sein. Zudem sollten langjährige Erfahrung in Minecraft mitbringen. Als letztes sollten sie freundlich und hilfsbereit sein.

Moderator

Ihre Aufgaben: Sie sorgen als Moderator bei uns dafür das die Regeln auf unseren Servern durchgesetzt werden. Außerdem sollten sie regelmäßig auf unser Server Netzwerk kommen. Das erwarten wir von ihnen: Sie haben ein Mikrofon und Lautsprecher und sind bereit die Software Teamspeak zu nutzen und scheuen sich nicht davor mit uns auf unserem Teamspeak Server zu sprechen. Zudem sind sie der deutschen Sprache im wörtlichen und schriftlichen mächtig. Außerdem sollten sie langjährige Erfahrung in Minecraft mitbringen. Sie sollten freundlich und hilfsbereit sein. Als letztes sollten sie auch verantwortungsbewusst sein.

So können sie sich Bewerben

Wenn sie sich die Texte oben durchgelesen haben und unsere anforderungen erfüllen, und beschlossen haben sich zu bewerben, Schicken sie einfach eine E-mail an Bewerben@Kampfläufermc.de zusammen mit ihrem Bewerbungs Schreiben. Wir melden uns auf jedenfall zurück.
