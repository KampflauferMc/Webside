Builder

Ihre Aufgaben: Sie bauen als Builder bei uns die Maps für unsere User. Zudem sollten sie regelmäßig auf unser Server Netzwerk kommen. Das erwarten wir von ihnen: Sie haben ein Mikrofon und Lautsprecher und sind bereit die Software Teamspeak zu nutzen und scheuen sich nicht davor mit uns auf unserem Teamspeak Server zu sprechen. Außerdem sollten sie der deutschen Sprache im wörtlichen und schriftlichen mächtig sein. Sie sollten langjährige Erfahrung in Minecraft mitbringen. Zudem sollten sie freundlich und hilfsbereit sein. Testaufgabe: Sie bauen in einer Minecraft Flat-World eine ……. nach und schicken diese/n an die E-Mail-Adresse Bewerben@Kampfläufermc.de zusammen mit ihrem Bewerbungsschreiben. Wenn uns ihre Bewerbung und der Minecraft Nachbau gefällt melden wir uns zurück und bieten ihnen ein persönliches Gespräch auf unserem Teamspeak Server an.

Developer

Ihre Aufgaben: Sie Programmieren in Java Minecraft Plugins. und sollten regelmäßig auf unser Server Netzwerk kommen Das erwarten wir von ihnen: Sie haben ein Mikrofon und Lautsprecher und sind bereit die Software Teamspeak zu nutzen und scheuen sich nicht davor mit uns auf unserem Teamspeak Server zu sprechen. Außerdem sollten sind der deutschen Sprache im wörtlichen und schriftlichen mächtig sein. Sie sollten langjährige Erfahrung in Minecraft mitbringen. Zudem sollten sich in der Programmiersprache Java auskennen. Wichtig ist auch das sie sich mit GitHub auskennen. Als letztes sollten sie freundlich und hilfsbereit sein. Testaufgabe: Als Beweis das sie sich mit Problemen in der Programmierung auseinandersetzten wollen und den nötigen Kampfgeist haben müssen sie als Aufgabe auf unserer Bewerben Website im F12 Modus versuchen einen Zugangscode zu kriegen. Wenn sie sich zu zweit bewerben können sie auch zusammenarbeiten. Nachdem sie den Zugangscode gefunden haben müssen sie noch eine Minecraft Plugin Programmieren das ……. Tut. Wenn sie beides gemacht haben, schicken sie eine E-Mail an die Adresse Bewerben@Kampfläufermc.de zusammen mit ihrem Bewerbungsschreiben, dem Plugin und dem Zugangscode. Wenn uns ihre Bewerbung und das Plugin gefällt und der Code richtig ist melden wir uns zurück und bieten ihnen ein persönliches Gespräch auf unserem Teamspeak Server an.

Das erwarten wir von ihnen: Sie haben ein Mikrofon und Lautsprecher und sind bereit die Software Teamspeak zu nutzen und scheuen sich nicht davor mit uns auf unserem Teamspeak Server zu sprechen. Außerdem sollten der deutschen Sprache im wörtlichen und schriftlichen mächtig sein. Zudem sollten langjährige Erfahrung in Minecraft mitbringen. Als letztes sollten sie freundlich und hilfsbereit sein. Testaufgabe: Als Testaufgabe sollten sie die unten aufgeführten Fragen beantworten: Was kann ich auf Kampfläufermc machen …… …… Wenn sie die Fragen beantwortet haben, schicken sie eine E-Mail an die Adresse Bewerben@Kampfläufermc.de zusammen mit ihren Antworten und einem Bewerbungsschreiben. Wenn uns ihre Bewerbung und die Beantwortung der Fragen gefällt melden wir uns zurück und bieten ihnen ein persönliches Gespräch auf unserem Teamspeak Server an.
